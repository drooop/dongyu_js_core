# Iterations

This directory stores iteration work packages under `docs/iterations/<id>/`.

Required files per iteration:
- `plan.md`
- `resolution.md`
- `runlog.md`

Use templates from `docs/_templates/` and register the iteration in `docs/ITERATIONS.md`
before execution. Optional `assets/` may store screenshots or other evidence.
