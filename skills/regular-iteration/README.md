# regular-iteration

本 skill 是仓库级“常规迭代（Regular Iteration）”的 Phase1 规划流程：只产出文档，不改代码。

## Purpose
- 对当前项目状态做一次结构化复盘。
- 提出一个可验证、可落地的新 Iteration，并生成 Phase1 文档。

## Usage Rules
- 遵守 `AGENTS.md` 与 `docs/WORKFLOW.md`。
- 只生成 Phase1 文档：`plan.md` / `resolution.md` / `runlog.md`。
- 停在 Review Gate，等待明确批准后才允许进入 Phase3。
